Sum All Elements in an Array

Topics Covered: Arrays, STL Algorithms, Iterators

Problem Statement

Write functions to calculate the sum of all elements in an array. Show multiple approaches:

  1. Using a manual loop (range-based for).
  2. Using std::accumulate from <numeric>.
  3. Using std::reduce from C++17 (parallel-capable).

Examples:

  • {1, 2, 3, 4, 5} → 15
  • {} → 0
  • {-1, 1, -1, 1} → 0

Manual Loop Approach

Click to reveal Solution

#include <iostream>
#include <vector>
#include <array>

// For std::vector
int sumVector(const std::vector<int>& nums) {
    int sum = 0;
    for (const auto& val : nums) {
        sum += val;
    }
    return sum;
}

// Template: works with any container of any numeric type
template<typename Container>
auto sumContainer(const Container& c) -> typename Container::value_type {
    using ValueType = typename Container::value_type;
    ValueType sum = ValueType{};
    for (const auto& val : c) {
        sum += val;
    }
    return sum;
}

// For C-style arrays (need the size explicitly)
template<typename T, std::size_t N>
T sumCArray(const T (&arr)[N]) {
    T sum = T{};
    for (std::size_t i = 0; i < N; ++i) {
        sum += arr[i];
    }
    return sum;
}

int main() {
    std::vector<int> nums = {1, 2, 3, 4, 5};
    std::cout << "Sum (manual loop): " << sumVector(nums) << "\n";  // 15

    int cArr[] = {10, 20, 30};
    std::cout << "Sum (C-array):    " << sumCArray(cArr) << "\n";     // 60

    std::vector<double> doubles = {1.5, 2.5, 3.0};
    std::cout << "Sum (template):   " << sumContainer(doubles) << "\n"; // 7.0

    return 0;
}

Explanation:

  • Range-based for loop is the most readable way to iterate in C++11 and later.
  • The template version deduces the return type from the container's value_type.
  • For C-style arrays, we use a template that captures the array size N automatically, avoiding the need for a separate size parameter.

Using std::accumulate

Click to reveal Solution

#include <iostream>
#include <vector>
#include <numeric>   // std::accumulate
#include <string>

int main() {
    std::vector<int> nums = {1, 2, 3, 4, 5};

    // Basic usage: sum with initial value 0
    int sum = std::accumulate(nums.begin(), nums.end(), 0);
    std::cout << "Sum: " << sum << "\n";  // 15

    // Note: passing 0 (int) as initial value computes int sum
    // For double precision, pass 0.0
    std::vector<double> doubles = {1.1, 2.2, 3.3};
    double dsum = std::accumulate(doubles.begin(), doubles.end(), 0.0);
    std::cout << "Double sum: " << dsum << "\n";  // 6.6

    // Concatenating strings with accumulate
    std::vector<std::string> words = {"Hello", " ", "World", "!"};
    std::string sentence = std::accumulate(words.begin(), words.end(), std::string{});
    std::cout << "Concatenated: " << sentence << "\n";  // "Hello World!"

    // Using a custom binary operation
    std::vector<int> vals = {2, 3, 4};
    int product = std::accumulate(vals.begin(), vals.end(), 1, std::multiplies<int>{});
    std::cout << "Product: " << product << "\n";  // 24

    return 0;
}

Explanation:

  • std::accumulate takes a range [begin, end) and an initial value.
  • The initial value determines both the type and the starting point. Passing 0 gives int accumulation; passing 0.0 gives double.
  • By default it uses operator+, but you can provide a custom binary operation as a fourth argument.
  • std::accumulate is not just for addition — you can use it for multiplication, string concatenation, or any fold operation.

Complexity Analysis:

  • Time: O(n) — every element is visited exactly once.
  • Space: O(1) — only the accumulator variable.

Follow-up Questions

Variations to Try

  1. Sum of even elements: Sum only elements that satisfy a condition (use std::accumulate with a lambda or combine with std::copy_if).
  2. Prefix sum array: Given an array, compute the running prefix sum where each element is the sum of all elements before it. Example: {1,2,3} → {0,1,3}.
  3. Sum of a 2D matrix: Compute the sum of all elements in a 2D vector (std::vector<std::vector<int>>).
  4. Parallel sum with std::reduce: Use C++17's std::reduce with an execution policy to sum a large vector in parallel across threads.

Key Takeaways:

  • std::accumulate (in <numeric>) is the idiomatic C++ way to fold/reduce a range into a single value.
  • Always match the initial value type to the desired result type: 0 for int, 0.0 for double, "" or std::string{} for strings.
  • The initial value is not just a starting point — it determines the accumulator's type, which affects precision for floating-point sums.
  • std::accumulate is a left fold: ((initial + a[0]) + a[1]) + a[2] .... The order matters for non-associative operations (like floating-point addition).
