Palindrome Check

Topics Covered: Strings, Arrays, Pointers

Problem Statement

Write a function that determines whether a given string is a palindrome. A palindrome reads the same forwards and backwards.

Examples:

  • "racecar" → true
  • "hello" → false
  • "A man a plan a canal Panama" → true (ignoring spaces and case)
  • "" → true
  • "a" → true

Requirements:

  • Implement a basic version that does an exact comparison.
  • Implement an enhanced version that ignores non-alphanumeric characters and is case-insensitive.
  • Use the two-pointer technique for O(1) space complexity.

Basic Palindrome Check

Click to reveal Solution

#include <iostream>
#include <string>

bool isPalindrome(const std::string& s) {
    int left = 0;
    int right = static_cast<int>(s.size()) - 1;

    while (left < right) {
        if (s[left] != s[right]) {
            return false;
        }
        ++left;
        --right;
    }
    return true;
}

int main() {
    std::cout << std::boolalpha;
    std::cout << "racecar: " << isPalindrome("racecar") << "\n";   // true
    std::cout << "hello:   " << isPalindrome("hello") << "\n";      // false
    std::cout << "empty:   " << isPalindrome("") << "\n";           // true
    std::cout << "a:       " << isPalindrome("a") << "\n";           // true
    std::cout << "ab:      " << isPalindrome("ab") << "\n";          // false
    std::cout << "aa:      " << isPalindrome("aa") << "\n";          // true
    return 0;
}

Explanation:

  • Two pointers start at opposite ends of the string and move toward the center.
  • If any pair of characters does not match, the string is not a palindrome.
  • If all pairs match (or the string has 0/1 characters), it is a palindrome.
  • Empty string and single-character strings are trivially palindromes.

Enhanced Version (Ignore Case and Non-alphanumeric)

Click to reveal Solution

#include <iostream>
#include <string>
#include <cctype>   // for std::isalnum, std::tolower

bool isPalindromeClean(const std::string& s) {
    int left = 0;
    int right = static_cast<int>(s.size()) - 1;

    while (left < right) {
        // Skip non-alphanumeric characters from the left
        while (left < right && !std::isalnum(static_cast<unsigned char>(s[left]))) {
            ++left;
        }
        // Skip non-alphanumeric characters from the right
        while (left < right && !std::isalnum(static_cast<unsigned char>(s[right]))) {
            --right;
        }

        // Compare characters case-insensitively
        if (std::tolower(static_cast<unsigned char>(s[left])) !=
            std::tolower(static_cast<unsigned char>(s[right]))) {
            return false;
        }
        ++left;
        --right;
    }
    return true;
}

int main() {
    std::cout << std::boolalpha;
    std::cout << "A man a plan a canal Panama: "
              << isPalindromeClean("A man a plan a canal Panama") << "\n";  // true
    std::cout << "racecar:  "
              << isPalindromeClean("racecar") << "\n";                        // true
    std::cout << "No lemon, no melon: "
              << isPalindromeClean("No lemon, no melon") << "\n";              // true
    std::cout << "hello:    "
              << isPalindromeClean("hello") << "\n";                             // false

    return 0;
}

Explanation:

  • Before each comparison, we skip any character that is not alphanumeric using std::isalnum.
  • Characters are compared after converting to lowercase with std::tolower.
  • The static_cast<unsigned char> is important because std::isalnum and std::tolower have undefined behavior if passed a negative char value.
  • This approach processes the string in a single pass without creating a cleaned copy, keeping space at O(1).

Complexity Analysis:

  • Time: O(n) — each character is visited at most once.
  • Space: O(1) — no extra storage beyond a few variables.

Follow-up Questions

Variations to Try

  1. Palindrome using std::equal: Use the standard library algorithm std::equal with reverse iterators to check for palindrome in a single line.
  2. Valid Palindrome II: Given a string, check if it is a palindrome after removing at most one character. Example: "abca" → true (remove 'b' or 'c').
  3. Longest palindromic substring: Find the longest substring within a string that is a palindrome. This is a classic medium-level problem.
  4. Number palindrome: Check if an integer is a palindrome without converting it to a string (e.g., 121 → true, -121 → false).

Common Pitfall:

When using <cctype> functions (std::isalnum, std::tolower, etc.), always cast char to unsigned char first. On platforms where char is signed, passing values like '\\x80' can trigger undefined behavior. This is a frequent source of bugs in production code.

Key Takeaways:

  • The two-pointer technique is ideal for palindrome checks: compare symmetric positions moving inward.
  • Always consider edge cases: empty strings, single characters, strings with only non-alphanumeric characters.
  • The enhanced version demonstrates how to process only relevant characters in-place, a useful pattern for text processing.
