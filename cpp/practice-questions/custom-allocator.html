Simple Pool Allocator

Topics Covered: Memory Management, Templates, Placement New

Problem Statement

Implement a simple pool allocator for fixed-size objects. The pool pre-allocates a contiguous block of memory and partitions it into equal-sized slots. Allocation returns a free slot instantly (O(1)); deallocation returns the slot to the free list (O(1)).

Requirements:

  • Template-based: works with any type T.
  • Fixed capacity specified at construction.
  • O(1) allocate and deallocate.
  • Detect double-free and out-of-memory conditions.
  • Use placement new for construction and explicit destructor calls for destruction.

#include <iostream>
#include <cstddef>
#include <new>        // placement new
#include <stdexcept>
#include <type_traits>
#include <vector>
#include <cassert>

template<typename T, std::size_t Capacity>
class PoolAllocator {
    // Align each slot to alignof(T)
    struct alignas(alignof(T)) Slot {
        std::byte storage[sizeof(T)];
    };

    // Free list node stored inside unused slots
    union FreeNode {
        FreeNode* next;
        // We don't need the full storage here since
        // FreeNode is only stored in unused slots
    };

public:
    PoolAllocator() {
        // Build the free list: each slot points to the next
        freeList_ = reinterpret_cast<FreeNode*>(&pool_[0]);
        for (std::size_t i = 0; i < Capacity - 1; ++i) {
            FreeNode* node = reinterpret_cast<FreeNode*>(&pool_[i]);
            node->next = reinterpret_cast<FreeNode*>(&pool_[i + 1]);
        }
        FreeNode* lastNode = reinterpret_cast<FreeNode*>(&pool_[Capacity - 1]);
        lastNode->next = nullptr;

        freeCount_ = Capacity;
        std::cout << "PoolAllocator: " << Capacity << " slots of "
                  << sizeof(T) << " bytes each\n";
    }

    ~PoolAllocator() {
        // All allocated objects should have been deallocated
        // In a real implementation, we might want to force-destroy
        if (freeCount_ != Capacity) {
            std::cerr << "Warning: " << (Capacity - freeCount_)
                      << " objects still allocated at destruction\n";
        }
    }

    // Non-copyable, non-movable
    PoolAllocator(const PoolAllocator&) = delete;
    PoolAllocator& operator=(const PoolAllocator&) = delete;

    // Allocate a slot and construct T with given arguments
    template<typename... Args>
    T* allocate(Args&&... args) {
        if (freeCount_ == 0) {
            throw std::bad_alloc();
        }

        // Pop from free list
        FreeNode* node = freeList_;
        freeList_ = node->next;
        --freeCount_;

        // Use placement new to construct T in the slot
        void* memory = static_cast<void*>(node);
        T* obj = new (memory) T(std::forward<Args>(args)...);
        return obj;
    }

    // Destroy and deallocate a slot
    void deallocate(T* obj) {
        if (!obj) return;

        // Verify the pointer is within our pool
        void* slotStart = static_cast<void*>(&pool_[0]);
        void* slotEnd   = static_cast<void*>(&pool_[Capacity]);
        void* ptr       = static_cast<void*>(obj);

        if (ptr < slotStart || ptr >= slotEnd) {
            throw std::invalid_argument("Pointer not from this pool");
        }

        // Explicitly call destructor
        obj->~T();

        // Push back to free list
        FreeNode* node = reinterpret_cast<FreeNode*>(ptr);
        node->next = freeList_;
        freeList_ = node;
        ++freeCount_;
    }

    std::size_t freeCount() const { return freeCount_; }
    std::size_t allocatedCount() const { return Capacity - freeCount_; }
    std::size_t capacity() const { return Capacity; }
    bool isEmpty() const { return freeCount_ == Capacity; }
    bool isFull() const { return freeCount_ == 0; }

private:
    alignas(alignof(T)) std::byte pool_[sizeof(Slot) * Capacity];
    FreeNode* freeList_ = nullptr;
    std::size_t freeCount_ = 0;
};

// --- Demo ---

struct Packet {
    int id;
    double timestamp;
    char data[32];

    Packet(int i, double t) : id(i), timestamp(t) {
        snprintf(data, sizeof(data), "Packet-%d", id);
    }
    ~Packet() {
        // std::cout << "Packet " << id << " destroyed\n";
    }
    void print() const {
        std::cout << "  Packet{id=" << id
                  << ", time=" << timestamp
                  << ", data='" << data << "'}\n";
    }
};

int main() {
    constexpr std::size_t POOL_SIZE = 8;
    PoolAllocator<Packet, POOL_SIZE> pool;

    std::cout << "Free slots: " << pool.freeCount() << "\n\n";

    // Allocate packets
    std::vector<Packet*> packets;
    for (int i = 0; i < 5; ++i) {
        Packet* p = pool.allocate(i, 1000.0 + i);
        packets.push_back(p);
        std::cout << "Allocated: "; p->print();
    }

    std::cout << "\nFree slots after 5 allocs: " << pool.freeCount() << "\n";

    // Deallocate some
    for (int i = 0; i < 3; ++i) {
        pool.deallocate(packets[i]);
        std::cout << "Deallocated packet " << i << "\n";
    }
    std::cout << "Free slots after 3 deallocs: " << pool.freeCount() << "\n";

    // Allocate again (reuses freed slots)
    Packet* p = pool.allocate(99, 9999.0);
    std::cout << "\nRe-allocated: "; p->print();

    // Cleanup remaining
    for (int i = 3; i < 5; ++i) {
        pool.deallocate(packets[i]);
    }
    pool.deallocate(p);
    std::cout << "\nFinal free slots: " << pool.freeCount() << " / " << pool.capacity() << "\n";

    return 0;
}

Explanation:

  • Pool memory: A contiguous array of Slot structs, each sized to hold a T and properly aligned using alignas.
  • Free list: Uses the unused slots themselves to store the linked list of free slots. The FreeNode::next pointer is written into the slot's memory when the slot is free.
  • Placement new: new (memory) T(args...) constructs a T object at a specific memory address without allocating memory. This separates allocation from construction.
  • Explicit destructor: obj->~T() calls the destructor without freeing memory. This separates destruction from deallocation.
  • Bounds checking: deallocate verifies that the pointer falls within the pool's memory range to catch double-free or wrong-pool errors.

Follow-up Questions

Variations to Try

  1. STL-compatible allocator: Adapt the pool allocator to satisfy the C++ Allocator requirements (rebind, allocate/deallocate with count, etc.) so you can use it with std::vector<T, PoolAlloc<T>>.
  2. Grow-only pool: Allow the pool to allocate additional chunks when it runs out of slots, instead of throwing bad_alloc.
  3. Multi-size pool: Create a memory pool that supports objects of different sizes by using separate pools for common size classes (8, 16, 32, 64, 128 bytes).
  4. Thread-safe pool: Add atomic operations or a lock to make the pool safe for concurrent allocation/deallocation.

Real-World Applications:

  • Pool allocators are widely used in embedded systems and telecom (e.g., 5G protocol stacks) where deterministic allocation time is critical.
  • Game engines use pool allocators for frequently created/destroyed objects like particles, bullets, or network packets.
  • The nginx web server uses memory pools extensively to avoid fragmentation.
  • The FreeRTOS RTOS provides xPoolCreate for exactly this pattern.
