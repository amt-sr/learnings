Implement a Singly Linked List

Topics Covered: Pointers, Classes, Memory Management, Constructors / Destructors

Problem Statement

Implement a complete singly linked list class in C++ with the following operations:

  • Insert: at the head, at the tail, and at a specific position.
  • Delete: by value, by position, and delete the entire list.
  • Search: find whether a value exists and return its position.
  • Reverse: reverse the linked list in-place.
  • Print: display all elements.

The implementation must properly manage memory with correct Rule of Five semantics.

Requirements:

  • Use proper RAII: the destructor must free all nodes.
  • Implement the Rule of Five (or Rule of Three if C++11 move semantics are not needed).
  • Handle edge cases: empty list, single element, inserting/deleting at boundaries.

#include <iostream>
#include <stdexcept>
#include <utility>

template<typename T>
class LinkedList {
private:
    struct Node {
        T data;
        Node* next;
        Node(const T& val) : data(val), next(nullptr) {}
        Node(T&& val) : data(std::move(val)), next(nullptr) {}
    };

    Node* head_ = nullptr;
    std::size_t size_ = 0;

public:
    // --- Rule of Five ---

    // Constructor
    LinkedList() = default;

    // Copy constructor
    LinkedList(const LinkedList& other) : size_(other.size_) {
        Node* current = other.head_;
        Node** tail = &head_;
        while (current) {
            *tail = new Node(current->data);
            tail = &((*tail)->next);
            current = current->next;
        }
    }

    // Move constructor
    LinkedList(LinkedList&& other) noexcept
        : head_(other.head_), size_(other.size_) {
        other.head_ = nullptr;
        other.size_ = 0;
    }

    // Copy assignment
    LinkedList& operator=(const LinkedList& other) {
        if (this != &other) {
            LinkedList temp(other);
            std::swap(head_, temp.head_);
            std::swap(size_, temp.size_);
        }
        return *this;
    }

    // Move assignment
    LinkedList& operator=(LinkedList&& other) noexcept {
        if (this != &other) {
            clear();
            head_ = other.head_;
            size_ = other.size_;
            other.head_ = nullptr;
            other.size_ = 0;
        }
        return *this;
    }

    // Destructor
    ~LinkedList() { clear(); }

    // --- Insertion ---

    void pushFront(const T& val) {
        Node* newNode = new Node(val);
        newNode->next = head_;
        head_ = newNode;
        ++size_;
    }

    void pushFront(T&& val) {
        Node* newNode = new Node(std::move(val));
        newNode->next = head_;
        head_ = newNode;
        ++size_;
    }

    void pushBack(const T& val) {
        if (!head_) {
            pushFront(val);
            return;
        }
        Node* current = head_;
        while (current->next) {
            current = current->next;
        }
        current->next = new Node(val);
        ++size_;
    }

    void insertAt(std::size_t index, const T& val) {
        if (index > size_) {
            throw std::out_of_range("Index out of range");
        }
        if (index == 0) {
            pushFront(val);
            return;
        }
        if (index == size_) {
            pushBack(val);
            return;
        }
        Node* current = head_;
        for (std::size_t i = 0; i < index - 1; ++i) {
            current = current->next;
        }
        Node* newNode = new Node(val);
        newNode->next = current->next;
        current->next = newNode;
        ++size_;
    }

    // --- Deletion ---

    void popFront() {
        if (!head_) throw std::out_of_range("List is empty");
        Node* temp = head_;
        head_ = head_->next;
        delete temp;
        --size_;
    }

    void popBack() {
        if (!head_) throw std::out_of_range("List is empty");
        if (!head_->next) {
            delete head_;
            head_ = nullptr;
            --size_;
            return;
        }
        Node* current = head_;
        while (current->next->next) {
            current = current->next;
        }
        delete current->next;
        current->next = nullptr;
        --size_;
    }

    bool removeByValue(const T& val) {
        if (!head_) return false;
        if (head_->data == val) {
            popFront();
            return true;
        }
        Node* current = head_;
        while (current->next) {
            if (current->next->data == val) {
                Node* temp = current->next;
                current->next = temp->next;
                delete temp;
                --size_;
                return true;
            }
            current = current->next;
        }
        return false;
    }

    bool removeAt(std::size_t index) {
        if (index >= size_) return false;
        if (index == 0) { popFront(); return true; }
        if (index == size_ - 1) { popBack(); return true; }

        Node* current = head_;
        for (std::size_t i = 0; i < index - 1; ++i) {
            current = current->next;
        }
        Node* temp = current->next;
        current->next = temp->next;
        delete temp;
        --size_;
        return true;
    }

    void clear() {
        Node* current = head_;
        while (current) {
            Node* next = current->next;
            delete current;
            current = next;
        }
        head_ = nullptr;
        size_ = 0;
    }

    // --- Search ---

    int find(const T& val) const {
        Node* current = head_;
        int index = 0;
        while (current) {
            if (current->data == val) return index;
            current = current->next;
            ++index;
        }
        return -1;  // Not found
    }

    bool contains(const T& val) const {
        return find(val) != -1;
    }

    // --- Reverse ---

    void reverse() {
        Node* prev = nullptr;
        Node* current = head_;
        while (current) {
            Node* next = current->next;
            current->next = prev;
            prev = current;
            current = next;
        }
        head_ = prev;
    }

    // --- Utility ---

    std::size_t size() const { return size_; }
    bool empty() const { return head_ == nullptr; }

    const T& front() const {
        if (!head_) throw std::out_of_range("List is empty");
        return head_->data;
    }

    const T& back() const {
        if (!head_) throw std::out_of_range("List is empty");
        Node* current = head_;
        while (current->next) current = current->next;
        return current->data;
    }

    void print() const {
        Node* current = head_;
        std::cout << "[";
        while (current) {
            std::cout << current->data;
            if (current->next) std::cout << " -> ";
            current = current->next;
        }
        std::cout << "] (size: " << size_ << ")\n";
    }
};

int main() {
    LinkedList<int> list;

    // Insert
    list.pushBack(1);
    list.pushBack(2);
    list.pushBack(3);
    list.pushFront(0);
    list.insertAt(2, 99);
    list.print();  // [0 -> 1 -> 99 -> 2 -> 3] (size: 5)

    // Search
    std::cout << "Find 99: index " << list.find(99) << "\n";  // 2
    std::cout << "Find 42: index " << list.find(42) << "\n";  // -1

    // Remove
    list.removeByValue(99);
    list.popFront();
    list.popBack();
    list.print();  // [1 -> 2] (size: 2)

    // Reverse
    list.pushBack(3);
    list.pushBack(4);
    list.print();  // [1 -> 2 -> 3 -> 4] (size: 4)
    list.reverse();
    list.print();  // [4 -> 3 -> 2 -> 1] (size: 4)

    // Copy
    LinkedList<int> copy = list;
    copy.print();  // [4 -> 3 -> 2 -> 1] (size: 4)

    return 0;
}

Explanation:

  • Node struct: Each node holds data and a pointer to the next node. The struct is nested inside the class to keep it encapsulated.
  • Rule of Five: We implement copy constructor, move constructor, copy assignment, move assignment, and destructor. The copy assignment uses the copy-and-swap idiom.
  • Insert: pushFront creates a new node and makes it the head. pushBack traverses to the end. insertAt traverses to the position before the target index.
  • Delete: The key pattern is keeping track of the previous node so you can update its next pointer after deletion. For head deletion, simply update head_.
  • Reverse: Three-pointer technique: prev, current, and next. On each iteration, we reverse the current->next pointer to point to prev, then advance all three pointers.

Follow-up Questions

Variations to Try

  1. Doubly linked list: Add a prev pointer to each node. This gives O(1) pushBack if you maintain a tail pointer, and O(n) reverse (or O(1) if bidirectional traversal is allowed).
  2. Remove duplicates: From a sorted linked list, remove all duplicate nodes in a single pass.
  3. Find the middle: Find the middle node using the slow/fast pointer technique (slow moves 1 step, fast moves 2 steps).
  4. Detect cycle: Implement Floyd's cycle detection algorithm (tortoise and hare).
  5. Merge two sorted lists: Given two sorted linked lists, merge them into one sorted list.

Memory Safety Tips:

  • Always delete nodes when removing them. Use valgrind to verify no leaks.
  • In the destructor, walk the entire list and delete every node. This is essential RAII.
  • After a move, set the source's head_ to nullptr to prevent double-free in the source's destructor.
  • Never dereference a null pointer. Always check head_ before operations.
