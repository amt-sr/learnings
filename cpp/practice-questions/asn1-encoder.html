BER-TLV Encoder/Decoder

Topics Covered: ASN.1, Byte Manipulation, Buffers

Problem Statement

Implement a BER-TLV (Basic Encoding Rules - Tag Length Value) encoder and decoder for a simple telecom message. BER-TLV is the encoding format used in SIM card communication, UICC, and various telecom protocols.

A BER-TLV data object consists of three parts:

  • Tag: Identifies the type of data (1 byte for simple tags, or multi-byte for extended).
  • Length: Number of bytes in the value field. Uses variable-length encoding.
  • Value: The actual data payload.

BER-TLV Format Reference:

Length byteMeaning
0x00 - 0x7FShort form: value is the length itself (0-127 bytes)
0x81Long form: next 1 byte is the length
0x82Long form: next 2 bytes (big-endian) are the length
0x83Long form: next 3 bytes (big-endian) are the length
0x84Long form: next 4 bytes (big-endian) are the length

// ---- TLV Data Structure ----

struct TlvElement {
    uint8_t tag = 0;
    std::vector<uint8_t> value;

    std::size_t totalSize() const {
        return 1 + encodeLengthSize(value.size()) + value.size();
    }

    // Convenience: interpret value as integer (first 4 bytes, big-endian)
    uint32_t toUint32() const {
        if (value.empty()) return 0;
        uint32_t result = 0;
        for (auto b : value) {
            result = (result << 8) | b;
        }
        return result;
    }

    // Convenience: interpret value as ASCII string
    std::string toString() const {
        return std::string(value.begin(), value.end());
    }
};

// ---- BER-TLV Encoder ----

class BerTlvEncoder {
public:
    // Encode a single TLV element into a byte buffer
    static std::vector<uint8_t> encode(const TlvElement& tlv) {
        std::vector<uint8_t> buffer;
        buffer.push_back(tlv.tag);                    // Tag (1 byte)
        encodeLength(buffer, tlv.value.size());        // Length
        buffer.insert(buffer.end(),
                      tlv.value.begin(),
                      tlv.value.end());                // Value
        return buffer;
    }

    // Encode multiple TLV elements into a single buffer
    static std::vector<uint8_t> encodeAll(
        const std::vector<TlvElement>& elements) {
        std::vector<uint8_t> buffer;
        for (const auto& elem : elements) {
            auto encoded = encode(elem);
            buffer.insert(buffer.end(), encoded.begin(), encoded.end());
        }
        return buffer;
    }

    // Helper: create TLV from integer (big-endian)
    static TlvElement fromUint8(uint8_t tag, uint8_t val) {
        return TlvElement{tag, {val}};
    }

    static TlvElement fromUint16(uint8_t tag, uint16_t val) {
        return TlvElement{tag, {
            static_cast<uint8_t>((val >> 8) & 0xFF),
            static_cast<uint8_t>(val & 0xFF)
        }};
    }

    static TlvElement fromUint32(uint8_t tag, uint32_t val) {
        return TlvElement{tag, {
            static_cast<uint8_t>((val >> 24) & 0xFF),
            static_cast<uint8_t>((val >> 16) & 0xFF),
            static_cast<uint8_t>((val >> 8) & 0xFF),
            static_cast<uint8_t>(val & 0xFF)
        }};
    }

    // Helper: create TLV from string
    static TlvElement fromString(uint8_t tag, const std::string& str) {
        return TlvElement{tag,
            std::vector<uint8_t>(str.begin(), str.end())};
    }

private:
    static void encodeLength(std::vector<uint8_t>& buffer, std::size_t length) {
        if (length < 0x80) {
            // Short form: single byte
            buffer.push_back(static_cast<uint8_t>(length));
        } else if (length <= 0xFF) {
            // Long form: 1 byte length
            buffer.push_back(0x81);
            buffer.push_back(static_cast<uint8_t>(length));
        } else if (length <= 0xFFFF) {
            // Long form: 2 byte length (big-endian)
            buffer.push_back(0x82);
            buffer.push_back(static_cast<uint8_t>((length >> 8) & 0xFF));
            buffer.push_back(static_cast<uint8_t>(length & 0xFF));
        } else {
            throw std::length_error("Length too large for simple BER encoding");
        }
    }

    static std::size_t encodeLengthSize(std::size_t length) {
        if (length < 0x80) return 1;
        if (length <= 0xFF) return 2;
        if (length <= 0xFFFF) return 3;
        return 5;
    }
};

// ---- BER-TLV Decoder ----

class BerTlvDecoder {
public:
    // Parse all TLV elements from a buffer
    static std::vector<TlvElement> decode(
        const uint8_t* data, std::size_t length) {
        std::vector<TlvElement> result;
        std::size_t offset = 0;

        while (offset < length) {
            if (offset >= length) {
                throw std::runtime_error("Incomplete TLV at offset " +
                    std::to_string(offset));
            }

            TlvElement tlv;
            tlv.tag = data[offset++];  // Read tag

            // Read length
            std::size_t valueLength = 0;
            if (offset >= length) {
                throw std::runtime_error("Missing length at offset " +
                    std::to_string(offset));
            }

            uint8_t firstLenByte = data[offset++];
            if (firstLenByte < 0x80) {
                // Short form
                valueLength = firstLenByte;
            } else if (firstLenByte == 0x81) {
                // Long form: 1 byte
                if (offset >= length) throw std::runtime_error("Incomplete length");
                valueLength = data[offset++];
            } else if (firstLenByte == 0x82) {
                // Long form: 2 bytes (big-endian)
                if (offset + 1 > length) throw std::runtime_error("Incomplete length");
                valueLength = (static_cast<std::size_t>(data[offset]) << 8) |
                              static_cast<std::size_t>(data[offset + 1]);
                offset += 2;
            } else {
                throw std::runtime_error("Unsupported length encoding: 0x" +
                    std::to_string(firstLenByte));
            }

            // Read value
            if (offset + valueLength > length) {
                throw std::runtime_error("Incomplete value: need " +
                    std::to_string(valueLength) + " bytes at offset " +
                    std::to_string(offset) + " but only " +
                    std::to_string(length - offset) + " available");
            }

            tlv.value.assign(data + offset, data + offset + valueLength);
            offset += valueLength;

            result.push_back(tlv);
        }

        return result;
    }

    // Find a specific TLV by tag
    static std::optional<TlvElement> findByTag(
        const std::vector<TlvElement>& elements, uint8_t tag) {
        for (const auto& elem : elements) {
            if (elem.tag == tag) return elem;
        }
        return std::nullopt;
    }
};

// ---- Hex dump helper ----

std::string hexDump(const std::vector<uint8_t>& data) {
    std::ostringstream oss;
    for (auto b : data) {
        oss << std::hex << std::setw(2) << std::setfill('0')
            << static_cast<int>(b) << " ";
    }
    return oss.str();
}

// ---- Demo ----

int main() {
    // Define tag names for readability
    // Example: a simple telecom message
    constexpr uint8_t TAG_IMSI          = 0x01;  // IMSI (string)
    constexpr uint8_t TAG_KSI           = 0x02;  // Key Set Identifier
    constexpr uint8_t TAG_SEQUENCE      = 0x03;  // Sequence number
    constexpr uint8_t TAG_AUTH_STATUS   = 0x04;  // Authentication status

    // Build a message
    std::vector<TlvElement> message;
    message.push_back(BerTlvEncoder::fromString(TAG_IMSI, "310260000000001"));
    message.push_back(BerTlvEncoder::fromUint8(TAG_KSI, 7));
    message.push_back(BerTlvEncoder::fromUint16(TAG_SEQUENCE, 42));
    message.push_back(BerTlvEncoder::fromUint8(TAG_AUTH_STATUS, 1));

    // Encode
    auto encoded = BerTlvEncoder::encodeAll(message);

    std::cout << "=== BER-TLV Encoder/Decoder Demo ===\n\n";
    std::cout << "Encoded (" << encoded.size() << " bytes):\n  ";
    std::cout << hexDump(encoded) << "\n\n";

    // Decode
    auto decoded = BerTlvDecoder::decode(encoded.data(), encoded.size());

    std::cout << "Decoded " << decoded.size() << TLV elements:\n";
    for (const auto& elem : decoded) {
        std::cout << "  Tag 0x" << std::hex << std::setw(2) << std::setfill('0')
                  << static_cast<int>(elem.tag) << std::dec;
        std::cout << ", Length=" << elem.value.size();
        std::cout << ", Value: ";

        // Try to display as string or integer
        if (elem.tag == TAG_IMSI) {
            std::cout << "\"" << elem.toString() << "\"";
        } else {
            std::cout << elem.toUint32();
        }
        std::cout << "\n";
    }

    // Find by tag
    std::cout << "\nLookup by tag:\n";
    auto ksi = BerTlvDecoder::findByTag(decoded, TAG_KSI);
    if (ksi) {
        std::cout << "  KSI: " << ksi->toUint32() << "\n";
    }

    auto missing = BerTlvDecoder::findByTag(decoded, 0xFF);
    if (!missing) {
        std::cout << "  Tag 0xFF: not found (as expected)\n";
    }

    // Round-trip: encode decoded back and compare
    auto reencoded = BerTlvEncoder::encodeAll(decoded);
    std::cout << "\nRound-trip match: "
              << (encoded == reencoded ? "YES" : "NO") << "\n";

    return 0;
}

Explanation:

  • Tag: Single byte identifying the data type. In full BER-TLV, tags can be multi-byte with class bits, constructed/primitive flag, and tag number. We use simple single-byte tags here.
  • Length encoding: Short form (1 byte, 0-127) and long form (first byte = 0x81/0x82/..., followed by 1-4 length bytes in big-endian). This is the BER-LER (Length Encoding Rules).
  • Encoder: Takes TlvElement structs, serializes each into Tag + Length + Value bytes, and concatenates them.
  • Decoder: Parses a byte buffer by reading Tag, decoding Length, then extracting Value bytes. Repeats until the buffer is consumed.
  • Bounds checking: Every read verifies sufficient data remains, throwing descriptive exceptions on malformed input.
  • Round-trip: Encode then decode produces identical data, verifying correctness.

Complexity Analysis:

  • Encode: O(n) where n is total bytes in all values (single pass).
  • Decode: O(n) single pass through the buffer.
  • Find by tag: O(k) where k is the number of TLV elements (linear scan).

Follow-up Questions

Variations to Try

  1. Nested TLV: Support constructed TLV where the value contains further TLV elements (recursive parsing). Tag bit 6 (constructed flag) = 1.
  2. Multi-byte tags: Support 2-byte tags where the first byte is 0xFF, indicating the tag number is in the next byte.
  3. DER encoding: Implement Distinguished Encoding Rules (DER) — a strict subset of BER that requires the shortest possible length encoding (no 0x82 for lengths that fit in 0x81).
  4. Template-based TLV builder: Create a fluent API for building TLV messages: message.add(IMSI, "310...").add(KSI, 7).build().
  5. Sim File System: Implement a parser for SIM EF (Elementary File) structures which use BER-TLV encoding extensively.

Real-World Context:

  • BER-TLV is the standard encoding for SIM/USIM card communication (APDUs), defined in TS 31.101 and TS 31.102.
  • 5G SUPI (Subscription Permanent Identifier) uses BCD encoding within TLV structures.
  • The UICC file system uses BER-TLV for file attributes and data objects.
  • Understanding TLV encoding is essential for any work involving SIM cards, smart cards, or telecom network signaling (MAP, CAP).
