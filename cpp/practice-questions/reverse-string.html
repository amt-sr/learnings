Reverse a String

Topics Covered: Strings, Pointers, Arrays

Problem Statement

Write a function that reverses a C++ std::string in-place without using the built-in std::reverse algorithm. Then, as a follow-up, show how to achieve the same result using std::reverse from <algorithm>.

Examples:

  • "hello" → "olleh"
  • "C++" → "++C"
  • "" → ""
  • "a" → "a"

Constraints:

  • Do it in-place (modify the original string, do not create a new one).
  • Handle edge cases: empty string, single character, odd/even length.
  • For the manual approach, use a two-pointer swap technique.

Manual Two-Pointer Approach

Click to reveal Solution

#include <iostream>
#include <string>

void reverseString(std::string& s) {
    int left = 0;
    int right = static_cast<int>(s.size()) - 1;

    while (left < right) {
        // Swap characters at left and right positions
        char temp = s[left];
        s[left] = s[right];
        s[right] = temp;
        ++left;
        --right;
    }
}

// Alternative: using std::swap
void reverseStringSwap(std::string& s) {
    int left = 0;
    int right = static_cast<int>(s.size()) - 1;

    while (left < right) {
        std::swap(s[left], s[right]);
        ++left;
        --right;
    }
}

int main() {
    std::string test1 = "hello";
    reverseString(test1);
    std::cout << "Reversed: " << test1 << "\n";  // "olleh"

    std::string test2 = "C++";
    reverseString(test2);
    std::cout << "Reversed: " << test2 << "\n";  // "++C"

    std::string test3 = "";
    reverseString(test3);
    std::cout << "Reversed: \"" << test3 << "\"\n";  // ""

    std::string test4 = "a";
    reverseString(test4);
    std::cout << "Reversed: " << test4 << "\n";  // "a"

    return 0;
}

Explanation:

  • We use two pointers: left starting at index 0 and right at the last index.
  • We swap the characters at these positions, then move left forward and right backward.
  • The loop stops when left >= right, meaning all pairs have been swapped.
  • For odd-length strings, the middle character is never swapped (it stays in place).
  • The static_cast<int> is needed because s.size() returns size_t (unsigned); subtracting 1 from 0 would overflow without the cast.

Complexity Analysis:

  • Time: O(n/2) = O(n) — we swap half the characters.
  • Space: O(1) — in-place, only a temp variable is used.

Using std::reverse (Follow-up)

Click to reveal Solution

#include <iostream>
#include <string>
#include <algorithm>  // for std::reverse

void reverseWithStd(std::string& s) {
    std::reverse(s.begin(), s.end());
}

int main() {
    std::string str = "hello world";
    reverseWithStd(str);
    std::cout << str << "\n";  // "dlrow olleh"
    return 0;
}

Explanation:

  • std::reverse is a standard library algorithm in <algorithm>.
  • It takes iterators (here s.begin() and s.end()) and reverses the range in-place.
  • Under the hood it uses the same two-pointer swap technique.
  • This is the idiomatic C++ approach — always prefer standard algorithms when they exist.

Follow-up Questions

Variations to Try

  1. Reverse only the vowels: Given a string, reverse only the vowel characters while keeping consonants in place. Example: "leetcode" → "leotcede".
  2. Reverse words in a string: Given "the sky is blue", return "blue is sky the". Handle multiple spaces and leading/trailing spaces.
  3. Reverse a C-string (char*): Implement the same logic for a null-terminated C-style string. This tests raw pointer arithmetic.
  4. Reverse a string recursively: Implement the reversal using recursion instead of iteration. What is the stack depth?

Key Takeaways:

  • The two-pointer technique is fundamental: one pointer moves forward, one backward, swapping elements. This pattern appears in palindrome checks, array rotations, and many other problems.
  • Always be careful with size_t vs int when doing arithmetic that involves subtraction (unsigned underflow).
  • In production code, prefer std::reverse over a manual loop — it is well-tested, expressive, and potentially optimized.
