std::thread

std::thread (C++11) provides a portable C++ interface for creating and managing OS threads. Each std::thread object represents a thread of execution. The C++ thread API abstracts away platform-specific threading (pthreads, Win32 threads) and integrates with the standard library's type system.

Creating and Joining Threads

#include <thread>
#include <iostream>

void hello() {
    std::cout << "Hello from thread " << std::this_thread::get_id() << "\n";
}

void thread_basics() {
    // Create a thread that runs hello()
    std::thread t(hello);

    // Wait for the thread to finish
    t.join();
    std::cout << "Main thread: " << std::this_thread::get_id() << "\n";
}

Passing Arguments

#include <thread>
#include <iostream>
#include <string>

void print_message(const std::string& msg, int count) {
    for (int i = 0; i < count; ++i) {
        std::cout << msg << " (" << i << ")\n";
    }
}

void passing_args_demo() {
    // Arguments are copied by default
    std::string msg = "Hello";
    std::thread t(print_message, msg, 3);

    // Use std::ref to pass by reference
    // std::thread t2(print_message, std::ref(msg), 3);

    t.join();
}

// Avoid dangling references: arguments are ALWAYS copied
// even if the thread function takes by reference!
void danger_demo() {
    std::string local = "hello";
    // BAD: passes a pointer to local variable that may be destroyed
    // std::thread t([](const std::string& s) { std::cout << s; }, local);
    // OK: pass by value or move
    std::thread t([local]() { std::cout << local << "\n"; });
    t.join();
}

Dangling reference trap: When passing arguments to std::thread, they are copied into internal storage and then passed to the function. If you pass a reference, the thread function receives a reference to the copy, not the original. Use std::ref() to pass by reference, or capture by value/move in a lambda.

Join vs Detach

#include <thread>
#include <iostream>

void join_vs_detach() {
    // join: wait for thread to finish (main thread blocks)
    std::thread t1([]() {
        std::this_thread::sleep_for(std::chrono::milliseconds(100));
        std::cout << "t1 finished\n";
    });
    t1.join();  // blocks until t1 completes
    std::cout << "After t1 join\n";

    // detach: thread runs independently (daemon thread)
    std::thread t2([]() {
        std::this_thread::sleep_for(std::chrono::milliseconds(50));
        std::cout << "t2 running independently\n";
    });
    t2.detach();  // t2 becomes a detached thread
    // Must ensure t2 does not access destroyed resources!
}

Always join or detach: If a std::thread object is destroyed without being joined or detached, std::terminate() is called. A common pattern is to use RAII to guarantee joining (see below).

RAII Thread Guard

#include <thread>

class ThreadGuard {
    std::thread& t_;
public:
    explicit ThreadGuard(std::thread& t) : t_(t) {}
    ~ThreadGuard() {
        if (t_.joinable()) {
            t_.join();
        }
    }
    ThreadGuard(const ThreadGuard&) = delete;
    ThreadGuard& operator=(const ThreadGuard&) = delete;
};

void raii_thread_demo() {
    std::thread t([]() {
        std::this_thread::sleep_for(std::chrono::milliseconds(100));
        std::cout << "Background work done\n";
    });

    ThreadGuard guard(t);  // auto-joins when scope exits
    // ... do other work ...
    // t is automatically joined when guard is destroyed
}

Parallel Vector Sum

#include <thread>
#include <vector>
#include <numeric>
#include <iostream>

void parallel_sum(const std::vector<int>& data,
                   size_t start, size_t end, long& result) {
    result = 0;
    for (size_t i = start; i < end; ++i) {
        result += data[i];
    }
}

void parallel_sum_demo() {
    std::vector<int> data(10'000'000);
    std::iota(data.begin(), data.end(), 1);  // 1 to 10M

    const size_t num_threads = std::thread::hardware_concurrency();
    std::vector<std::thread> threads;
    std::vector<long> partial_sums(num_threads);

    size_t chunk = data.size() / num_threads;

    for (size_t i = 0; i < num_threads; ++i) {
        size_t start = i * chunk;
        size_t end = (i == num_threads - 1) ? data.size() : start + chunk;
        threads.emplace_back(parallel_sum,
            std::cref(data), start, end, std::ref(partial_sums[i]));
    }

    for (auto& t : threads) t.join();

    long total = 0;
    for (auto s : partial_sums) total += s;

    std::cout << "Parallel sum: " << total << "\n";
    // Expected: 50000005000000
}

Common Thread Operations

OperationDescription
t.joinable()Returns true if thread can be joined
t.join()Waits for thread to finish
t.detach()Runs thread independently
t.get_id()Returns thread ID
std::this_thread::get_id()ID of the calling thread
std::this_thread::sleep_for(dur)Sleep for a duration
std::this_thread::sleep_until(tp)Sleep until a time point
