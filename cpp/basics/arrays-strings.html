1.1.6 Arrays & Strings

Arrays store collections of elements of the same type in contiguous memory. Strings are sequences of characters, and C++ provides both C-style strings and the modern std::string class.

C-Style Arrays

C-style arrays have a fixed size determined at compile time. They are simple but lack bounds checking and do not know their own length.

#include <iostream>

int main() {
    // Declaration and initialization
    int nums[5] = {10, 20, 30, 40, 50};

    // Partial initialization (remaining elements are zero-initialized)
    int partial[5] = {1, 2};  // {1, 2, 0, 0, 0}

    // Size deduction from initializer
    int deduced[] = {100, 200, 300};  // int[3]

    // Access elements (zero-indexed)
    std::cout << "First: " << nums[0] << std::endl;
    std::cout << "Last: " << nums[4] << std::endl;

    // Determine number of elements
    int count = sizeof(nums) / sizeof(nums[0]);
    std::cout << "Element count: " << count << std::endl;

    // Iterate
    for (int i = 0; i < count; ++i) {
        std::cout << nums[i] << " ";
    }
    std::cout << std::endl;

    return 0;
}

Warning: C-style arrays do not perform bounds checking. Accessing nums[100] when the array has 5 elements is undefined behavior — it will read whatever happens to be in that memory location. This is a common source of crashes and security vulnerabilities. Prefer std::array or std::vector for safer alternatives.

std::array (C++11)

std::array is a fixed-size container from the standard library. It wraps a C-style array with safety features and utility methods.

#include <iostream>
#include <array>

int main() {
    std::array<int, 5> arr = {10, 20, 30, 40, 50};

    // Bounds-checked access (throws std::out_of_range)
    std::cout << arr.at(0) << std::endl;

    // Size is always available
    std::cout << "Size: " << arr.size() << std::endl;

    // Range-based for loop
    for (const auto& elem : arr) {
        std::cout << elem << " ";
    }
    std::cout << std::endl;

    // Fill with a value
    std::array<int, 5> zeros{};
    zeros.fill(42);
    for (const auto& elem : zeros) {
        std::cout << elem << " ";
    }
    std::cout << std::endl;

    return 0;
}

Tip: Prefer std::array over C-style arrays. It knows its own size, supports .at() for bounds checking, works with range-based for loops, and has the same performance as raw arrays (zero overhead).

Multi-dimensional Arrays

#include <iostream>

int main() {
    // 3x3 matrix (C-style)
    int matrix[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    // Access element at row 1, column 2
    std::cout << "matrix[1][2] = " << matrix[1][2] << std::endl;

    // Iterate over all elements
    for (int i = 0; i < 3; ++i) {
        for (int j = 0; j < 3; ++j) {
            std::cout << matrix[i][j] << "\t";
        }
        std::cout << std::endl;
    }

    return 0;
}

C-Strings

C-style strings are null-terminated character arrays. They are inherited from C and still appear frequently in C++ code, especially in system programming and when interfacing with C libraries.

#include <iostream>
#include <cstring>  // for strlen, strcpy, strcmp

int main() {
    // C-string: character array terminated by '\0'
    char greeting[] = "Hello";  // Automatically null-terminated

    // Manual C-string
    char name[20];
    std::strcpy(name, "Alice");

    std::cout << greeting << std::endl;
    std::cout << "Length: " << std::strlen(greeting) << std::endl;

    // Comparison (returns 0 if equal)
    if (std::strcmp(greeting, "Hello") == 0) {
        std::cout << "Strings match" << std::endl;
    }

    return 0;
}

Warning: strcpy and strcat do not check buffer sizes and can cause buffer overflows. Use strncpy and strncat with explicit size limits, or better yet, use std::string which handles memory management automatically.

std::string

std::string is the modern C++ string class. It manages memory automatically, provides safe operations, and integrates with the STL. See std::string Deep Dive for comprehensive coverage.

#include <iostream>
#include <string>

int main() {
    // Construction
    std::string s1 = "Hello";
    std::string s2("World");
    std::string s3(5, 'x');  // "xxxxx"

    // Concatenation
    std::string greeting = s1 + ", " + s2 + "!";
    std::cout << greeting << std::endl;

    // Length and capacity
    std::cout << "Length: " << greeting.length() << std::endl;
    std::cout << "Size: " << greeting.size() << std::endl;

    // Substring
    std::string sub = greeting.substr(0, 5);  // "Hello"
    std::cout << "Substring: " << sub << std::endl;

    // Find
    size_t pos = greeting.find("World");
    if (pos != std::string::npos) {
        std::cout << "Found 'World' at position " << pos << std::endl;
    }

    // Comparison
    std::cout << (s1 == "Hello" ? "equal" : "not equal") << std::endl;

    // Iteration
    for (char c : greeting) {
        std::cout << c;
    }
    std::cout << std::endl;

    return 0;
}

String Stream (Parsing)

std::stringstream is useful for converting between strings and numeric types, and for parsing delimited data.

#include <iostream>
#include <sstream>
#include <string>

int main() {
    // String to int
    std::stringstream ss("42 3.14 hello");

    int num;
    double pi;
    std::string word;

    ss >> num >> pi >> word;

    std::cout << "num = " << num << std::endl;
    std::cout << "pi = " << pi << std::endl;
    std::cout << "word = " << word << std::endl;

    // Number to string
    std::ostringstream oss;
    oss << "Score: " << 95 << " / 100";
    std::string result = oss.str();
    std::cout << result << std::endl;

    return 0;
}
