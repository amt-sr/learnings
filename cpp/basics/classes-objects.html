1.2.1 Classes & Objects

A class is a user-defined type that bundles data (member variables) and behavior (member functions) into a single unit. An object is an instance of a class. Classes are the foundation of object-oriented programming in C++ and enable encapsulation, inheritance, and polymorphism.

Class Definition Syntax

A class is defined using the class keyword followed by the class name and a body enclosed in braces. By default, all members of a class are private. A semicolon must follow the closing brace:

#include <iostream>
#include <string>

class Person {
private:                  // only accessible within the class
    std::string name;
    int age;

public:                   // accessible from outside the class
    // Constructor (initializes the object)
    Person(const std::string& n, int a) : name(n), age(a) {}

    // Member function (method)
    void greet() const {
        std::cout << "Hi, I'm " << name
                  << " and I'm " << age << " years old." << std::endl;
    }

    // Getter
    int get_age() const { return age; }

    // Setter
    void set_age(int a) {
        if (a >= 0) {
            age = a;
        }
    }
};  // <-- semicolon is required

Access Specifiers

C++ provides three access specifiers to control visibility of class members:

SpecifierAccess from classAccess from derivedAccess from outside
publicYesYesYes
protectedYesYesNo
privateYesNoNo

Default access: Members of a class are private by default. Members of a struct are public by default. In all other ways, class and struct are equivalent in C++.

Creating Objects

Once a class is defined, you create objects (instances) of that class:

#include <iostream>

int main() {
    // Create objects using the constructor
    Person alice{"Alice", 30};
    Person bob{"Bob", 25};

    // Call member functions using the dot operator
    alice.greet();   // Hi, I'm Alice and I'm 30 years old.
    bob.greet();     // Hi, I'm Bob and I'm 25 years old.

    // Use setters and getters
    alice.set_age(31);
    std::cout << "Alice is now " << alice.get_age() << std::endl;

    // Create a pointer to an object
    Person* ptr = &alice;
    ptr->greet();  // use arrow operator with pointers

    // Create a reference to an object
    Person& ref = bob;
    ref.greet();   // use dot operator with references

    return 0;
}

Tip: Use the . (dot) operator to access members through objects and references. Use the -> (arrow) operator to access members through pointers.

Member Variables and Member Functions

Member variables (also called fields or attributes) hold the state of an object. Member functions (also called methods) define the behavior. Member functions defined inside the class body are implicitly inline:

class Rectangle {
private:
    double width;
    double height;

public:
    Rectangle(double w, double h) : width(w), height(h) {}

    // Defined inside the class (implicitly inline)
    double area() const {
        return width * height;
    }

    // Defined outside the class (needs Rectangle:: scope)
    double perimeter() const;
};

// Member function defined outside the class
double Rectangle::perimeter() const {
    return 2.0 * (width + height);
}

The this Pointer

Inside every non-static member function, the keyword this is an implicit pointer to the object on which the function was called. It is used to disambiguate between member variables and parameters with the same name:

class Account {
private:
    double balance;

public:
    // Without 'this': parameter name must differ from member
    Account(double initial_balance) : balance(initial_balance) {}

    // With 'this': parameter can have the same name
    void deposit(double balance) {
        this->balance += balance;
        // 'this->balance' = member variable
        // 'balance'       = parameter
    }

    void display() const {
        std::cout << "Balance: $" << this->balance << std::endl;
    }

    // Returning *this enables method chaining
    Account& set_balance(double b) {
        balance = b;
        return *this;  // return reference to the current object
    }
};

int main() {
    Account acc{1000.0};
    acc.deposit(500.0);
    acc.display();  // Balance: $1500

    // Method chaining
    acc.set_balance(2000.0).deposit(300.0);
    acc.display();  // Balance: $2300

    return 0;
}

Warning: In a const member function, this has type const ClassName* — you cannot modify members through it. This enforces the promise that a const method does not change the object's state.

Complete Example

#include <iostream>
#include <cmath>

class Circle {
private:
    double radius;

public:
    Circle(double r) : radius(r) {
        if (radius <= 0) radius = 1.0;
    }

    double area() const { return 3.14159265 * radius * radius; }
    double circumference() const { return 2.0 * 3.14159265 * radius; }

    void scale(double factor) {
        if (factor > 0) radius *= factor;
    }

    // Accessor
    double get_radius() const { return radius; }

    // Mutator with validation
    void set_radius(double r) {
        if (r > 0) radius = r;
    }

    void print() const {
        std::cout << "Circle(r=" << radius
                  << ", area=" << area()
                  << ", circ=" << circumference()
                  << ")" << std::endl;
    }
};

int main() {
    Circle c{5.0};
    c.print();        // Circle(r=5, area=78.5398, circ=31.4159)

    c.scale(2.0);
    c.print();        // Circle(r=10, area=314.159, circ=62.8319)

    return 0;
}
