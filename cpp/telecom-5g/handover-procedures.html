Handover Procedures

Handover is the process of transferring a UE's connection from one cell to another. In LTE, handover is network-controlled (the eNodeB decides based on UE measurement reports). There are intra-LTE handovers (same RAN) and inter-RAT handovers (different radio technology, e.g., LTE to 3G).

Handover Phases

1. Preparation Phase

  • UE sends Measurement Report (RSRP/RSRQ of neighbor cells)
  • Source eNodeB evaluates A3 event (neighbor becomes better than serving + offset)
  • Source sends X2 Handover Request to target eNodeB with UE context
  • Target reserves resources, replies with Handover Ack (target RRC config)

2. Execution Phase

  • Source sends RRC Connection Reconfiguration with mobility control info
  • UE synchronizes to target cell via RACH
  • UE sends RRC Reconfiguration Complete to target
  • Source forwards buffered PDCP SDUs to target via X2-U

3. Completion Phase

  • Target sends Path Switch Request via S1 to MME
  • MME updates S1-U path (S-GW switches DL tunnel to target)
  • Source releases UE context after timeout or UE Context Release

Packet loss: During X2 handover, some packets may be lost during the gap between the UE disconnecting from source and connecting to target. PDCP handles duplicate detection and reordering. For seamless handover, the中断时间 target must be < 50ms for voice (VoLTE) to maintain quality.

Conditional Handover (CHO): In 5G, the source gNB can pre-configure the UE with multiple candidate target cells. The UE executes handover autonomously when conditions are met, reducing preparation latency to near-zero for the actual switch.

C++ Handover State Machine

#include <cstdint>

enum class HoState : uint8_t {
    IDLE,
    PREPARATION,
    EXECUTION,
    COMPLETION,
    CANCELLED
};

enum class HoType : uint8_t {
    INTRA_ENB,       // Same eNodeB
    X2_BASED,        // X2 inter-eNodeB
    S1_BASED,        // S1 inter-eNodeB (no X2)
    INTER_RAT        // LTE to/from 3G/2G
};

struct HandoverContext {
    uint32_t ue_id;
    uint32_t source_cell_id;
    uint32_t target_cell_id;
    HoType   ho_type;
    HoState  state;
    uint32_t ho_id;
    bool     rrc_sent;
    bool     data_forwarding_active;
};

// Handover manager at source eNodeB
class HandoverManager {
public:
    // Initiate handover based on measurement report
    void initiateHandover(uint32_t ue_id,
                           uint32_t target_cell_id) {
        auto& ctx = ho_contexts_[ue_id];
        ctx.ue_id = ue_id;
        ctx.target_cell_id = target_cell_id;
        ctx.state = HoState::PREPARATION;

        // Determine handover type
        if (isSameEnb(ctx.source_cell_id, target_cell_id)) {
            ctx.ho_type = HoType::INTRA_ENB;
        } else if (hasX2Interface(target_cell_id)) {
            ctx.ho_type = HoType::X2_BASED;
        } else {
            ctx.ho_type = HoType::S1_BASED;
        }

        // Send handover request to target
        if (ctx.ho_type == HoType::X2_BASED) {
            sendX2HandoverRequest(ctx);
        } else if (ctx.ho_type == HoType::S1_BASED) {
            sendS1HandoverRequest(ctx);
        }
    }

    void onHandoverAck(uint32_t ue_id) {
        auto it = ho_contexts_.find(ue_id);
        if (it == ho_contexts_.end()) return;

        auto& ctx = it->second;
        ctx.state = HoState::EXECUTION;

        // Forward buffered data to target
        startDataForwarding(ctx);

        // Send RRC HO command to UE
        sendRrcHandoverCommand(ue_id);
        ctx.rrc_sent = true;
    }

    void onRrcHandoverComplete(uint32_t ue_id) {
        auto it = ho_contexts_.find(ue_id);
        if (it == ho_contexts_.end()) return;
        it->second.state = HoState::COMPLETION;
    }

    void cancelHandover(uint32_t ue_id) {
        auto it = ho_contexts_.find(ue_id);
        if (it == ho_contexts_.end()) return;
        it->second.state = HoState::CANCELLED;
    }

private:
    std::unordered_map<uint32_t, HandoverContext> ho_contexts_;

    bool isSameEnb(uint32_t src, uint32_t tgt) { return false; }
    bool hasX2Interface(uint32_t cell_id) { return true; }
    void sendX2HandoverRequest(HandoverContext&) {}
    void sendS1HandoverRequest(HandoverContext&) {}
    void startDataForwarding(HandoverContext&) {}
    void sendRrcHandoverCommand(uint32_t) {}
};
