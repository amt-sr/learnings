Telecom Practice: Build a Mini 5G Protocol Stack

Topics Covered: 5G RAN Overview, CU-DU Split, MAC Layer, RLC Layer, PDCP Layer, RRC Layer, SDAP Layer, NAS Layer, LTE Architecture, Handover Procedures, Fixed-Point Arithmetic, Circular Buffers, Memory Pools, Packet Processing (DPDK), ASN.1 Encoding, Bit-Level Operations, Signal Processing Basics

Problem Statement

Implement a simplified protocol stack that processes packets through RLC -> PDCP -> RRC layers, mirroring the 5G NR user-plane and control-plane processing chain. The stack should handle downlink encoding (SDU in, PDU out) and uplink decoding (PDU in, SDU out) through each layer, demonstrating the layered architecture used in real gNodeB implementations.

This exercise is broken into 7 incremental sub-problems. Each builds on the previous one, so solve them in order.

Design Considerations:

  • In a real 5G stack, packets flow downlink as: Application -> SDAP -> PDCP -> RLC -> MAC -> PHY. We simplify this to RRC -> PDCP -> RLC -> MAC for the control plane.
  • Each layer adds its own header (and optionally trailer) to the SDU from the upper layer, creating a PDU that the lower layer treats as its SDU.
  • RLC handles segmentation when the SDU is too large for the MAC grant.
  • PDCP handles header compression (ROHC) and integrity/ciphering for control-plane data.
  • The MAC layer uses a circular buffer (ring buffer) to manage scheduling grants efficiently.

1. Define Protocol Layer Interfaces

Define protocol layer interfaces

Create an abstract ProtocolLayer base class that all layers inherit from. It should provide:

  • processDownlink(SDU) -> PDU: encode data going from upper layer down to lower layer (transmit path).
  • processUplink(PDU) -> SDU: decode data coming from lower layer up to upper layer (receive path).
  • A pure virtual destructor for proper cleanup via base pointers.

Also define an enum class LayerType identifying each layer (RRC, PDCP, RLC, MAC, NAS). A Packet struct holds raw bytes and metadata (layer type, sequence number).

2. Implement the RLC Layer

Implement the RLC layer

Implement RlcAmLayer (RLC Acknowledged Mode) that inherits from ProtocolLayer. It must support:

  • Segmentation: If the SDU exceeds a configurable maximum PDU size, split it into multiple PDUs. Each PDU gets a 2-byte RLC header containing a sequence number (12 bits) and segmentation info (4 bits).
  • Reassembly: On the uplink, reassemble segmented PDUs back into a single SDU using sequence numbers.
  • Sequence number tracking: Maintain a TX counter (next SN to transmit) and RX counter (next expected SN). Wrap around at 4096 (12-bit space).

// ---- RLC AM Header (2 bytes) ----
// Byte 0: [D/C=1 | P=0 | SN high 10 bits ]
// Byte 1: [SN low 2 bits | SO high 6 bits ]
// Byte 2: [SO low 8 bits ]  (only for continuation segments)
// For simplicity we use a 2-byte header:
//   Bits [15:14] = segment type (00=first, 01=continuation, 10=last, 11=complete)
//   Bits [13:2]  = sequence number (12 bits)
//   Bits [1:0]   = reserved

struct RlcHeader {
    uint16_t raw;

    enum SegType : uint8_t { FIRST = 0, CONT = 1, LAST = 2, COMPLETE = 3 };

    RlcHeader(SegType seg, uint16_t sn)
        : raw((static_cast<uint16_t>(seg) << 14) |
              ((sn & 0x0FFF) << 2)) {}

    SegType segType() const {
        return static_cast<SegType>((raw >> 14) & 0x03);
    }
    uint16_t sn() const { return (raw >> 2) & 0x0FFF; }

    std::vector<uint8_t> toBytes() const {
        return {static_cast<uint8_t>(raw >> 8),
                static_cast<uint8_t>(raw & 0xFF)};
    }

    static RlcHeader fromBytes(const uint8_t* b) {
        uint16_t r = (static_cast<uint16_t>(b[0]) << 8) | b[1];
        return RlcHeader(r);
    }
private:
    RlcHeader(uint16_t r) : raw(r) {}
};

class RlcAmLayer : public ProtocolLayer {
public:
    static constexpr std::size_t DEFAULT_MAX_PDU = 64; // bytes
    static constexpr uint16_t SN_MODULUS = 4096;       // 12-bit

    explicit RlcAmLayer(std::size_t maxPduSize = DEFAULT_MAX_PDU)
        : maxPduSize_(maxPduSize),
          txSn_(0), rxSn_(0) {}

    // Downlink: segment SDU into one or more PDUs
    // Returns a vector of PDUs (one SDU may produce multiple PDUs)
    std::vector<Packet> segmentDownlink(const Packet& sdu) {
        std::vector<Packet> pdus;
        const auto& payload = sdu.data;
        std::size_t payloadPerPdu = (maxPduSize_ >= 2)
                                    ? maxPduSize_ - 2 : 1;
        std::size_t offset = 0;

        if (payload.size() <= payloadPerPdu) {
            // Fits in one PDU
            Packet pdu;
            pdu.sourceLayer = layerType();
            pdu.destLayer = LayerType::MAC;
            pdu.sequenceNumber = txSn_;
            RlcHeader hdr(RlcHeader::COMPLETE, txSn_);
            pdu.prepend(hdr.toBytes());
            pdu.append(payload);
            pdus.push_back(std::move(pdu));
            txSn_ = (txSn_ + 1) % SN_MODULUS;
        } else {
            // Needs segmentation
            uint16_t sn = txSn_;
            while (offset < payload.size()) {
                std::size_t remaining = payload.size() - offset;
                std::size_t chunk = std::min(remaining, payloadPerPdu);
                Packet pdu;
                pdu.sourceLayer = layerType();
                pdu.destLayer = LayerType::MAC;
                pdu.sequenceNumber = sn;

                RlcHeader::SegType segType;
                if (offset == 0) segType = RlcHeader::FIRST;
                else if (offset + chunk >= payload.size()) segType = RlcHeader::LAST;
                else segType = RlcHeader::CONT;

                RlcHeader hdr(segType, sn);
                pdu.prepend(hdr.toBytes());
                pdu.data.insert(pdu.data.end(),
                                payload.begin() + offset,
                                payload.begin() + offset + chunk);
                pdus.push_back(std::move(pdu));
                offset += chunk;
            }
            txSn_ = (txSn_ + 1) % SN_MODULUS;
        }
        return pdus;
    }

    // Simplified single-PDU downlink (for stack compatibility)
    Packet processDownlink(const Packet& sdu) override {
        auto pdus = segmentDownlink(sdu);
        // If only one PDU, return it; otherwise concatenate
        if (pdus.size() == 1) return std::move(pdus[0]);
        Packet combined;
        combined.sourceLayer = layerType();
        combined.destLayer = LayerType::MAC;
        for (auto& pdu : pdus) {
            combined.data.insert(combined.data.end(),
                                 pdu.data.begin(), pdu.data.end());
        }
        return combined;
    }

    // Uplink: reassemble PDUs into an SDU
    Packet processUplink(const Packet& pdu) override {
        if (pdu.data.size() < 2) {
            return {};  // Too short for header
        }
        RlcHeader hdr = RlcHeader::fromBytes(pdu.data.data());
        std::vector<uint8_t> payload(pdu.data.begin() + 2,
                                      pdu.data.end());

        if (hdr.segType() == RlcHeader::COMPLETE) {
            // Single PDU - no reassembly needed
            rxSn_ = (hdr.sn() + 1) % SN_MODULUS;
            Packet sdu;
            sdu.data = std::move(payload);
            sdu.destLayer = LayerType::PDCP;
            return sdu;
        }

        // Multi-segment reassembly
        reassemblyBuffer_[hdr.sn()].push_back(
            {hdr.segType(), std::move(payload)});

        auto& segments = reassemblyBuffer_[hdr.sn()];
        bool hasFirst = false, hasLast = false;
        for (const auto& seg : segments) {
            if (seg.type == RlcHeader::FIRST)  hasFirst = true;
            if (seg.type == RlcHeader::LAST)   hasLast = true;
        }

        if (hasFirst && hasLast) {
            // Sort: FIRST, CONT(s), LAST
            std::sort(segments.begin(), segments.end(),
                [](const auto& a, const auto& b) {
                    return static_cast<int>(a.type) <
                           static_cast<int>(b.type);
                });

            Packet sdu;
            sdu.destLayer = LayerType::PDCP;
            for (const auto& seg : segments) {
                sdu.data.insert(sdu.data.end(),
                                seg.payload.begin(),
                                seg.payload.end());
            }
            reassemblyBuffer_.erase(hdr.sn());
            rxSn_ = (hdr.sn() + 1) % SN_MODULUS;
            return sdu;
        }

        return {};  // Incomplete, waiting for more segments
    }

    LayerType layerType() const override { return LayerType::RLC; }
    std::string name() const override { return "RLC-AM"; }

private:
    struct Segment {
        RlcHeader::SegType type;
        std::vector<uint8_t> payload;
    };

    std::size_t maxPduSize_;
    uint16_t txSn_;
    uint16_t rxSn_;
    std::map<uint16_t, std::vector<Segment>> reassemblyBuffer_;
};

Explanation:

  • RLC Header (2 bytes): Encodes segment type (FIRST/CONT/LAST/COMPLETE) in 2 bits and a 12-bit sequence number. This mirrors the real 5G NR RLC header structure (simplified).
  • Segmentation: segmentDownlink() splits large SDUs into chunks that fit within maxPduSize_. The first segment is tagged FIRST, intermediate ones CONT, and the last one LAST. Complete (non-segmented) PDUs get the COMPLETE tag.
  • Reassembly buffer: Segments are stored in a std::map keyed by SN. When both FIRST and LAST segments arrive, all segments are sorted and concatenated into the original SDU.
  • Sequence number wraparound: TX/RX counters wrap at 4096 (12-bit space), matching the real RLC SN space in 5G NR.

3. Implement the PDCP Layer

Implement the PDCP layer

Implement PdcpLayer that inherits from ProtocolLayer. It must support:

  • Header compression: Simulate ROHC (Robust Header Compression) by replacing repeated byte patterns with a 1-byte token. Maintain a small dictionary of seen patterns.
  • Integrity protection: Simulate by appending a 4-byte XOR-based checksum over the PDCP header and payload.
  • Sequence number management: 18-bit PDCP SN (used in 5G NR), with a 1-byte PDCP header containing SN[17:10] and SN[9:2] in two bytes (simplified to a 2-byte header with a 12-bit SN for this exercise).

// ---- PDCP Header (2 bytes) ----
// Byte 0: [D/C=1 | R=0 | SN[11:4]]
// Byte 1: [SN[3:0] | reserved 4 bits]
// Simplified: 12-bit SN with 4-bit reserved

struct PdcpHeader {
    uint16_t raw;

    PdcpHeader(uint16_t sn)
        : raw((sn & 0x0FFF) << 4) {}

    uint16_t sn() const { return (raw >> 4) & 0x0FFF; }

    std::vector<uint8_t> toBytes() const {
        return {static_cast<uint8_t>(raw >> 8),
                static_cast<uint8_t>(raw & 0xFF)};
    }

    static PdcpHeader fromBytes(const uint8_t* b) {
        return PdcpHeader(static_cast<uint16_t>(b[0]) << 8 | b[1]);
    }
private:
    PdcpHeader(uint16_t r) : raw(r) {}
};

// ---- ROHC-like header compression simulator ----

class SimulatedRohc {
public:
    static constexpr std::size_t PATTERN_LEN = 3;
    static constexpr uint8_t TOKEN_BASE = 0xE0;  // tokens 0xE0..0xFF

    // Compress: replace known 3-byte patterns with a 1-byte token
    static std::vector<uint8_t> compress(
        const std::vector<uint8_t>& data,
        std::map<std::vector<uint8_t>, uint8_t>& dict,
        uint8_t& nextToken) {
        if (data.size() < PATTERN_LEN) return data;

        std::vector<uint8_t> result;
        std::size_t i = 0;
        while (i < data.size()) {
            if (i + PATTERN_LEN <= data.size()) {
                std::vector<uint8_t> pattern(
                    data.begin() + i,
                    data.begin() + i + PATTERN_LEN);
                auto it = dict.find(pattern);
                if (it != dict.end()) {
                    result.push_back(it->second);
                    i += PATTERN_LEN;
                    continue;
                }
            }
            result.push_back(data[i]);
            ++i;
        }
        return result;
    }

    // Decompress: replace tokens back with patterns
    static std::vector<uint8_t> decompress(
        const std::vector<uint8_t>& data,
        const std::map<uint8_t, std::vector<uint8_t>>& revDict) {
        std::vector<uint8_t> result;
        for (auto b : data) {
            if (b >= TOKEN_BASE) {
                auto it = revDict.find(b);
                if (it != revDict.end()) {
                    result.insert(result.end(),
                                  it->second.begin(),
                                  it->second.end());
                    continue;
                }
            }
            result.push_back(b);
        }
        return result;
    }

    static void learnPattern(
        const std::vector<uint8_t>& pattern,
        std::map<std::vector<uint8_t>, uint8_t>& dict,
        std::map<uint8_t, std::vector<uint8_t>>& revDict,
        uint8_t& nextToken) {
        if (dict.find(pattern) == dict.end()) {
            dict[pattern] = nextToken;
            revDict[nextToken] = pattern;
            ++nextToken;
        }
    }
};

// ---- Simulated integrity protection (XOR checksum) ----

std::vector<uint8_t> computeChecksum(
    const std::vector<uint8_t>& header,
    const std::vector<uint8_t>& payload) {
    uint32_t xorSum = 0;
    for (auto b : header) xorSum ^= static_cast<uint32_t>(b);
    for (auto b : payload) xorSum ^= static_cast<uint32_t>(b);
    return {
        static_cast<uint8_t>((xorSum >> 24) & 0xFF),
        static_cast<uint8_t>((xorSum >> 16) & 0xFF),
        static_cast<uint8_t>((xorSum >> 8)  & 0xFF),
        static_cast<uint8_t>(xorSum & 0xFF)
    };
}

// ---- PDCP Layer ----

class PdcpLayer : public ProtocolLayer {
public:
    static constexpr uint16_t SN_MODULUS = 4096; // 12-bit

    PdcpLayer() : txSn_(0), rxSn_(0), nextToken_(0xE0) {}

    Packet processDownlink(const Packet& sdu) override {
        // 1. Learn patterns from the SDU for future compression
        if (sdu.data.size() >= SimulatedRohc::PATTERN_LEN) {
            for (std::size_t i = 0;
                 i + SimulatedRohc::PATTERN_LEN <= sdu.data.size();
                 i += SimulatedRohc::PATTERN_LEN) {
                std::vector<uint8_t> pat(
                    sdu.data.begin() + i,
                    sdu.data.begin() + i + SimulatedRohc::PATTERN_LEN);
                SimulatedRohc::learnPattern(
                    pat, compressDict_, decompressDict_, nextToken_);
            }
        }

        // 2. Header compression
        auto compressed = SimulatedRohc::compress(
            sdu.data, compressDict_, nextToken_);

        // 3. Build PDCP header
        PdcpHeader hdr(txSn_);
        auto hdrBytes = hdr.toBytes();

        // 4. Compute integrity checksum
        auto macI = computeChecksum(hdrBytes, compressed);

        // 5. Assemble PDU: header + compressed payload + MAC-I
        Packet pdu;
        pdu.sourceLayer = layerType();
        pdu.destLayer = LayerType::RLC;
        pdu.sequenceNumber = txSn_;
        pdu.data = hdrBytes;
        pdu.data.insert(pdu.data.end(),
                        compressed.begin(), compressed.end());
        pdu.data.insert(pdu.data.end(),
                        macI.begin(), macI.end());

        txSn_ = (txSn_ + 1) % SN_MODULUS;
        return pdu;
    }

    Packet processUplink(const Packet& pdu) override {
        if (pdu.data.size() < 6) {  // 2 hdr + 4 mac-i minimum
            return {};
        }

        // 1. Parse header
        PdcpHeader hdr = PdcpHeader::fromBytes(pdu.data.data());
        auto hdrBytes = hdr.toBytes();

        // 2. Extract payload and MAC-I
        std::vector<uint8_t> compressed(
            pdu.data.begin() + 2,
            pdu.data.end() - 4);
        std::vector<uint8_t> receivedMacI(
            pdu.data.end() - 4,
            pdu.data.end());

        // 3. Verify integrity
        auto computedMacI = computeChecksum(hdrBytes, compressed);
        bool integrityOk = (receivedMacI == computedMacI);
        if (!integrityOk) {
            std::cout << "PDCP: Integrity check FAILED for SN="
                      << hdr.sn() << "\n";
        }

        // 4. Header decompression
        auto decompressed = SimulatedRohc::decompress(
            compressed, decompressDict_);

        // 5. Build SDU
        Packet sdu;
        sdu.data = std::move(decompressed);
        sdu.destLayer = LayerType::RRC;
        sdu.sequenceNumber = hdr.sn();
        rxSn_ = (hdr.sn() + 1) % SN_MODULUS;
        return sdu;
    }

    LayerType layerType() const override { return LayerType::PDCP; }
    std::string name() const override { return "PDCP"; }

private:
    uint16_t txSn_;
    uint16_t rxSn_;
    uint8_t nextToken_;
    std::map<std::vector<uint8_t>, uint8_t> compressDict_;
    std::map<uint8_t, std::vector<uint8_t>> decompressDict_;
};

  • ROHC simulation: The SimulatedRohc class maintains a dictionary of 3-byte patterns. On the first pass, patterns are learned. On subsequent passes, repeated patterns are replaced with 1-byte tokens (0xE0+), achieving a 3:1 compression ratio for repeated patterns.
  • Integrity protection: A 4-byte XOR checksum is computed over the header and compressed payload, then appended as MAC-I. On decode, the checksum is recomputed and verified. In real 5G, this uses AES-based algorithms (SNOW 3G or AES-CMAC).
  • Compression is stateful: The second processDownlink call produces a shorter PDU because the IP header pattern was learned from the first call.

// ---- TLV encoding helpers ----

struct TlvField {
    uint8_t tag;
    std::vector<uint8_t> value;

    static TlvField fromUint8(uint8_t t, uint8_t v) {
        return {t, {v}};
    }
    static TlvField fromUint16(uint8_t t, uint16_t v) {
        return {t, {
            static_cast<uint8_t>((v >> 8) & 0xFF),
            static_cast<uint8_t>(v & 0xFF)
        }};
    }
    static TlvField fromUint32(uint8_t t, uint32_t v) {
        return {t, {
            static_cast<uint8_t>((v >> 24) & 0xFF),
            static_cast<uint8_t>((v >> 16) & 0xFF),
            static_cast<uint8_t>((v >> 8) & 0xFF),
            static_cast<uint8_t>(v & 0xFF)
        }};
    }
    static TlvField fromBytes(uint8_t t,
                               const std::vector<uint8_t>& v) {
        return {t, v};
    }

    uint32_t toUint32() const {
        uint32_t r = 0;
        for (auto b : value) r = (r << 8) | b;
        return r;
    }
    uint16_t toUint16() const {
        return static_cast<uint16_t>(toUint32() & 0xFFFF);
    }
};

// Encode TLV: Tag(1) + Length(1) + Value(N)
std::vector<uint8_t> encodeTlv(const TlvField& f) {
    std::vector<uint8_t> out = {f.tag,
        static_cast<uint8_t>(f.value.size() & 0xFF)};
    out.insert(out.end(), f.value.begin(), f.value.end());
    return out;
}

// Decode TLV fields from a byte buffer
std::vector<TlvField> decodeTlv(const uint8_t* data,
                                  std::size_t len) {
    std::vector<TlvField> fields;
    std::size_t off = 0;
    while (off + 2 <= len) {
        uint8_t tag = data[off++];
        uint8_t length = data[off++];
        if (off + length > len) break;
        TlvField f;
        f.tag = tag;
        f.value.assign(data + off, data + off + length);
        fields.push_back(std::move(f));
        off += length;
    }
    return fields;
}

// RRC Header (1 byte): upper 4 bits = msg type, lower 4 bits = length / 4

struct RrcHeader {
    uint8_t raw;

    RrcHeader(RrcMsgType type, std::size_t payloadLen)
        : raw((static_cast<uint8_t>(type) << 4) |
              static_cast<uint8_t>((payloadLen + 3) / 4 & 0x0F)) {}

    RrcMsgType msgType() const {
        return static_cast<RrcMsgType>((raw >> 4) & 0x0F);
    }
    uint8_t lengthUnits() const { return raw & 0x0F; }
    std::size_t payloadLength() const {
        return static_cast<std::size_t>(lengthUnits()) * 4;
    }
};

template<typename T>
class CircularBuffer {
public:
    explicit CircularBuffer(std::size_t capacity)
        : buffer_(capacity), head_(0), tail_(0), count_(0) {}

    bool push(const T& item) {
        if (count_ == buffer_.size()) return false;  // Full
        buffer_[tail_] = item;
        tail_ = (tail_ + 1) % buffer_.size();
        ++count_;
        return true;
    }

    bool push(T&& item) {
        if (count_ == buffer_.size()) return false;
        buffer_[tail_] = std::move(item);
        tail_ = (tail_ + 1) % buffer_.size();
        ++count_;
        return true;
    }

    std::optional<T> pop() {
        if (count_ == 0) return std::nullopt;  // Empty
        T item = std::move(buffer_[head_]);
        head_ = (head_ + 1) % buffer_.size();
        --count_;
        return item;
    }

    std::optional<T> peek() const {
        if (count_ == 0) return std::nullopt;
        return buffer_[head_];
    }

    std::size_t size() const { return count_; }
    std::size_t capacity() const { return buffer_.size(); }
    bool empty() const { return count_ == 0; }
    bool full() const { return count_ == buffer_.size(); }

private:
    std::vector<T> buffer_;
    std::size_t head_;
    std::size_t tail_;
    std::size_t count_;
};

        return pdu;

            scheduler.allocateGrant(pdu.size());
        }
        // Fill grants from PDUs
        scheduler.fillFromRlc(rlcPdus);
    }

private:

};

// PDCP Layer (simplified: header + checksum, no compression)
// ============================================================

class PdcpLayer : public ProtocolLayer {
public:
    Packet processDownlink(const Packet& sdu) override {
        Packet pdu = sdu;
        // PDCP header: 2 bytes with SN
        uint16_t sn = txSn_++;
        auto hdr = std::vector<uint8_t>{
            static_cast<uint8_t>((sn >> 8) | 0x80),
            static_cast<uint8_t>(sn & 0xFF)
        };
        pdu.prepend(hdr);

        // PDCP trailer: 4-byte MAC-I (simulated XOR checksum)
        uint32_t xorSum = 0;
        for (auto b : pdu.data) xorSum ^= b;
        pdu.data.push_back(static_cast<uint8_t>((xorSum >> 24) & 0xFF));
        pdu.data.push_back(static_cast<uint8_t>((xorSum >> 16) & 0xFF));
        pdu.data.push_back(static_cast<uint8_t>((xorSum >> 8) & 0xFF));
        pdu.data.push_back(static_cast<uint8_t>(xorSum & 0xFF));

        pdu.sourceLayer = layerType();
        pdu.destLayer = LayerType::RLC;
        pdu.sequenceNumber = sn;
        return pdu;
    }

    Packet processUplink(const Packet& pdu) override {
        if (pdu.data.size() < 6) return {};  // 2 hdr + 4 mac-i

        Packet sdu;
        sdu.data.assign(pdu.data.begin() + 2,
                         pdu.data.end() - 4);
        sdu.sourceLayer = LayerType::PDCP;
        sdu.destLayer = LayerType::RRC;
        uint16_t sn = (static_cast<uint16_t>(pdu.data[0] & 0x7F) << 8) |
                       static_cast<uint16_t>(pdu.data[1]);
        sdu.sequenceNumber = sn;

        // Verify MAC-I
        uint32_t xorSum = 0;
        for (std::size_t i = 0; i < pdu.data.size() - 4; ++i) {
            xorSum ^= pdu.data[i];
        }
        uint32_t receivedMac = (static_cast<uint32_t>(pdu.data[pdu.data.size()-4]) << 24) |
                                 (static_cast<uint32_t>(pdu.data[pdu.data.size()-3]) << 16) |
                                 (static_cast<uint32_t>(pdu.data[pdu.data.size()-2]) << 8)  |
                                 static_cast<uint32_t>(pdu.data[pdu.data.size()-1]);
        std::cout << "    [PDCP] MAC-I verify: "
                  << (xorSum == receivedMac ? "OK" : "FAIL") << "\n";

        return sdu;
    }

    LayerType layerType() const override { return LayerType::PDCP; }
    std::string name() const override { return "PDCP"; }

private:
    uint16_t txSn_ = 0;
};

class MacLayer : public ProtocolLayer {
public:
    explicit MacLayer(std::size_t maxSize = 128) : maxSize_(maxSize) {}

    Packet processDownlink(const Packet& sdu) override {
        Packet pdu = sdu;
        // MAC subheader: 1 byte (LCID=1 for SRB1)
        pdu.prepend({0x01});

        // Pad to max size if needed (simulating TTI boundary)
        if (pdu.size() < maxSize_) {
            std::size_t padding = maxSize_ - pdu.size();
            pdu.data.resize(maxSize_, 0x00);
        }

        pdu.sourceLayer = layerType();
        pdu.destLayer = LayerType::PHY;
        pdu.sequenceNumber = ttiCount_++;
        return pdu;
    }

    Packet processUplink(const Packet& pdu) override {
        if (pdu.data.empty()) return {};

        Packet sdu;
        sdu.data.assign(pdu.data.begin() + 1, pdu.data.end());

        // Strip trailing padding (zeros)
        while (!sdu.data.empty() && sdu.data.back() == 0x00) {
            sdu.data.pop_back();
        }

        sdu.sourceLayer = LayerType::MAC;
        sdu.destLayer = LayerType::RLC;
        std::cout << "    [MAC] Stripped padding, LCID="
                  << static_cast<int>(pdu.data[0]) << "\n";
        return sdu;
    }

    LayerType layerType() const override { return LayerType::MAC; }
    std::string name() const override { return "MAC"; }

private:
    std::size_t maxSize_;
    uint32_t ttiCount_ = 0;
};

Explanation:

  • End-to-end flow: A NAS Registration Request message (14 bytes) is encoded through all four layers, producing a 64-byte MAC PDU. Each layer adds its own header and/or trailer. The MAC layer pads to 64 bytes (simulating a fixed-size TTI grant).
  • Layer overhead breakdown: RRC (+1 byte), PDCP (+2 header + 4 trailer), RLC (+2 bytes), MAC (+1 byte + padding). Total overhead: 10 bytes of protocol headers + 40 bytes of padding.
  • Uplink decoding: Each layer strips its own header in reverse order. MAC removes padding, RLC strips its 2-byte header, PDCP verifies the MAC-I and strips header+trailer, RRC strips its 1-byte header, yielding the original NAS message.
  • Integrity verification: PDCP's uplink path recomputes the XOR checksum and compares it against the received MAC-I, logging "OK" or "FAIL".
  • Round-trip correctness: The recovered NAS message exactly matches the original, confirming the stack implementation is consistent.

Follow-up Questions

Variations to Try

  1. RLC segmentation with MAC grant awareness: Modify the RLC layer to accept a grant size parameter and segment the SDU to fit within the available grant. The RLC should buffer untransmitted segments and resume on the next grant.
  2. PDCP ciphering: Add a simple XOR cipher to the PDCP layer using a key derived from a KNAS/KNR security context. The cipher is applied after header compression but before the MAC-I is computed.
  3. Multi-UE stack: Extend the ProtocolStack to manage multiple UE contexts, each with its own layer instances, sequence numbers, and security contexts. Use a map keyed by UE ID (CRNTI).
  4. Timer-based RLC retransmission: Add a retransmission timer to the RLC AM layer. When a PDU is transmitted, start a timer. If no ACK is received before expiry, retransmit from the buffer.

Key Takeaways:

  • Layered architecture: The protocol stack follows the fundamental design principle of layered communication: each layer has a single responsibility, adds its own header, and treats the upper layer's output as its input (SDU). This is the same pattern used in OSI, TCP/IP, and 3GPP protocol stacks.
  • Downlink = prepend headers, Uplink = strip headers: On the transmit path, each layer prepends its header (and appends trailer for PDCP MAC-I). On the receive path, each layer strips what it added. This symmetric design ensures correct decoding.
  • Sequence numbers are per-layer: RLC SN (12-bit), PDCP SN (12-bit in our simplified version, 18-bit in real 5G NR), and RRC transaction IDs are independent counters. Each layer tracks its own state.
  • Segmentation happens at RLC: When MAC grants are smaller than the PDCP PDU, the RLC layer segments. Reassembly at the receiver requires buffering out-of-order segments and sorting by sequence number.
  • Circular buffers for scheduling: The MAC scheduler uses a ring buffer to manage grants efficiently. Lock-free ring buffers are a critical data structure in real-time telecom systems, enabling zero-copy packet processing under tight deadline constraints.
  • Integrity and ciphering at PDCP: Security functions (header compression via ROHC, integrity protection via MAC-I, and ciphering) are all performed at the PDCP layer. Understanding this placement is essential for 5G security architecture.
  • ASN.1 TLV encoding at RRC: RRC messages use ASN.1 BER-TLV encoding (defined in 3GPP TS 38.331). The Tag-Length-Value pattern is a universal building block for structured binary protocols in telecom.
