What to Learn

Definition

A Functional Requirement specifies what a system must do.

Examples:

  • User can register.
  • User can login.
  • User can upload a file.
  • User can search.
  • User can send messages.

Functional Requirement Checklist

Whenever you're given a design problem, identify:

  • Actors
  • Features
  • Inputs
  • Outputs
  • State Changes
  • External Systems
  • Business Rules

Skip

Do not spend time on:

  • IEEE SRS document templates
  • UML Use Case diagrams
  • Formal requirements engineering processes
  • Enterprise requirement management tools
  • Requirements traceability matrices

Those are valuable in software engineering practice but have very low ROI for senior system design interviews.
